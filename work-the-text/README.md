# Задача по HTML и CSS: работа с тегами для оформления текста и стилизация текста используя CSS.

Выполните все пункты задания и сравните с результатом. Для поиска неизвестных вам тегов и свойств используйте поиск справочника.

1. Создайте папку в удобном для вас месте на вашем компьютере<br>
2. В этой папке создайте новый HTML документ - index.html<br>
3. В index.html создайте HTML скелет документа<br>
4. Создайте новый CSS файл - style.css<br>
5. Подключите CSS файл к HTML файлу<br>
6. Создайте заголовок первого уровня в теге body и напишите там текст "Оформление текста"<br>
7. Добавьте данному заголовку класс title<br>
8. В CSS файле в самом верху создайте селектор для тега body и напишите следующие стили - шрифт Arial, sans-serif, размер шрифта 16px, цвет текста #333, межстрочный отступ 1.5<br>
9. В CSS файле создайте селектор для класса title, и напишите следующие стили - размер шрифта 40px, цвет текста #f03333, межстрочный отступ 1.2, все буквы заглавные<br>
10. После заголовка создайте абзац и напишите там немного текста, можете использовать сайт-генератор случайного текста lipsum.com<br>
11. После абзаца создайте заголовок второго уровня, напишите текст "Заголовок второго уровня" и придайте ему класс subtitle<br>
12. В CSS файле создайте селектор для класса subtitle, и напишите следующие стили - размер шрифта 30px, цвет текста #12ac11, межстрочный отступ 1.2, подчеркивание текста снизу<br>
13. После заголовка создайте абзац и напишите там немного текста, можете использовать сайт-генератор случайного текста lipsum.com<br>
14. После абзаца создайте ненумерованный список с тремя пунктами<br>
15. В каждом пункте напишите немного текста, на свой выбор<br>
16. Задайте списку класс list<br>
17. В CSS файле создайте селектор для класса list, и напишите следующие стили - размер шрифта 20px, цвет текста #444, все буквы наклонные, стиль маркеров списка - square
